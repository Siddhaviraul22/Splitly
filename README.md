# Splitly

Splitly is an advanced full-stack expense sharing web application with AI-powered insights.

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React
- Database: MySQL
- AI Service: Python (Flask, scikit-learn)

## Features (Planned)
- User authentication
- Group expense management
- Expense splitting (equal, exact, percentage)
- Balance tracking and settlement
- Analytics dashboards
- AI-based expense insights and predictions
